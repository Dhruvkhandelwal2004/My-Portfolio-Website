# Personal Portfolio Website

This is a modern and fully responsive personal portfolio website built using HTML5, CSS3, and Bootstrap 5.  
The website is designed to showcase my skills, projects, services, and resume in a clean and professional layout.

## 🚀 Features

- Fully Responsive Design (Mobile-First Approach)
- Modern Dark Theme with Accent Styling
- Multi-Page Structure
- Projects Showcase Section
- Skills Section with Progress Bars
- Services Section
- Resume Page
- Contact Form with Social Media Links
- Clean and Structured UI Layout

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons  

## 📂 Project Structure

- `home.html` – Landing Page  
- `about.html` – About Me Section  
- `skills.html` – Technical Skills  
- `projects.html` – Project Showcase  
- `services.html` – Services Offered  
- `resume.html` – Resume Details  
- `contact.html` – Contact Form  

## 🎯 Objective

The main objective of this project is to demonstrate frontend development skills, responsive design principles, and structured multi-page website architecture using Bootstrap.
